# Budget App

Персональный трекер бюджета. React + Vite.

## Быстрый старт

```bash
cd budget-app
npm install
npm run dev
```

Открой http://localhost:5173

## Структура

```
src/
  components/
    Dashboard.jsx   — дашборд с метриками и сводкой
    Entry.jsx       — ввод данных за месяц
    History.jsx     — история и график
    Settings.jsx    — управление счетами и источниками дохода
  utils/
    storage.js      — localStorage, миграция из старых версий, утилиты
  App.jsx           — роутинг между экранами
  main.jsx          — точка входа
  index.css         — стили
```

## Данные

Хранятся в `localStorage` под ключом `budget_app`.  
При первом запуске автоматически мигрируют из старых ключей `budget_v3`, `budget_v2`.

Экспорт в JSON — кнопка в разделе «Настройки».
